# Simple banking system project on hyperskill

----

**In a nutshell, this project is a simple banking system with menu, that has following items:**

```
1. Create an account
2. Log into account
3. Exit
```

**and after you logging in:**

```
1. Balance
2. Add income
3. Do transfer
4. Close account
5. Log out
6. Exit
```

----

**Used technologies:** _Java, SQLite, Gradle._
